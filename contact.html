Procedurally Generated Dungeon Simulation
Java

A video game where the player can explore procedurally generated top-down worlds. In the world, the player can only see the tiles around him, and must navigate the dungeon to find the unlocked door. This project made use of serialization, 2D array manipulation, and persistence. It also implemented saving and loading, mouse tracking, and a custom vision algorithm to only show viewable tiles.

Mapping Platform
Java
Mapping Image

Created a mapping platform based on the data provided by openstreetmap.org for the Berkeley area. Implemented rastering of the map for resizing, A* Algorithm for finding the shortest path to locations on the map, and created an extrinsic min priority queue for the A* algorithm.
